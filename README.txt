Shneska001:ANDRO:WifiDataGrabber is Android application designed for Wifi scanning and putting all needed intel alongside with GPS information.
Only for Androids below 6.0!

This app was designed to collect data about wifi hotspots on wider range. Therefore you need to physically move to collect wifi datas.
It grabs BSSID, SSID, Security strings, GPS position. It scans and saves the wifispots in sharedprefs file. With collected data you can for example
create whole map of wifi hotspots in city along with their security options, MAC addresses and locations. It can also detect hidden SSID networks.
You can use it for any purpouse data collecting.
